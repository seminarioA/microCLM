# microCLM

CRM de pipeline comercial migrado de HTML/CSS/JS vanilla a **React + TypeScript + Vite**, con la identidad visual de **La Segunda Mordida (LSM)** y un lenguaje de diseño minimalista de inspiración japonesa (espacio en blanco generoso, líneas finas, color usado como acento puntual).

## Módulos

- **Tablero Kanban** — pipeline de ventas en 8 etapas con drag & drop.
- **Captación de Leads** — formulario con validación y simulación de enriquecimiento por IA.
- **Dashboard Gerencial** — KPIs, embudo de conversión y gráficos (Chart.js).
- **Perfil del Cliente** — datos del cliente y timeline de interacciones con acciones rápidas.

## Stack

- React 19 + TypeScript
- Vite
- Chart.js / react-chartjs-2
- lucide-react (iconografía)

## Desarrollo

```bash
npm install
npm run dev
```
